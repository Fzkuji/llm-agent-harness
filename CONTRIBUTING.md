# Contributing to Agentic Programming

Thanks for your interest in contributing! This project is a paradigm proposal with a reference implementation — we welcome all kinds of contributions.

## Ways to Contribute

- **Use it and share feedback** — Try the paradigm in your own projects and tell us what works (or doesn't)
- **Report bugs** — Open an issue with reproduction steps
- **Suggest features** — Open an issue describing your use case
- **Submit PRs** — Bug fixes, new providers, examples, docs improvements
- **Build your own implementation** — In any language! We'd love to link to it

## Development Setup

```bash
git clone https://github.com/Fzkuji/Agentic-Programming.git
cd Agentic-Programming
pip install -e ".[dev]"
pytest
```

## Pull Request Guidelines

1. Fork the repo and create a branch from `main`
2. Add tests for new functionality
3. Ensure all tests pass (`pytest`)
4. Keep PRs focused — one feature or fix per PR

## Code Style

- Follow existing code patterns
- Use type hints
- Write docstrings for public APIs

## Questions?

Open a [Discussion](https://github.com/Fzkuji/Agentic-Programming/discussions) or an issue. We're happy to help!
